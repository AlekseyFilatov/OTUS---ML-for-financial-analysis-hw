Домашнее задание

ДЗ №8 «Выходим в мир»

Цель:

Студенты учатся подключаться к торговой площадке и использовать API для автоматизации действий.

Цель - атоматизация действий на торговой площадке.


Описание/Пошаговая инструкция выполнения домашнего задания:

 - создать тестовый аккаунт на торговой площадке;
 - подключить API торговой площадки к модели;
 - проверить на demo аккаунте, что ваша модель успешно выполняет торговые действия.
