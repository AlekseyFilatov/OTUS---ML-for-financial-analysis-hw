Домашнее задание

ДЗ №9 «Production окружение»

Цель:

Студенты учатся подключать созданную модель к API торговой площадки для выполнения торговых действий.

Цель - организовать работу модели в режиме 7х24 в облачной среде.


Описание/Пошаговая инструкция выполнения домашнего задания:
 - Создать необходимую инфраструктуру на одной из облачных платформ
 - Развернуть вашу модель
 - Обеспечить поступление данных через конвейер предобработки и формирование
рекомендации по торговому действию
 - Настроить связь вашей модели с одной из торговых площадок. Так же настроить
ограничения, чтобы не потерять значительных средств, в случае если ваша модель будет
рекомендовать не оптимальные действия.

Дополнительно:

 - Сформировать дашборд, показывающий текущую ситуацию и эффективность
вашей торговой стратегии
 - Настроить оповещения о показателях работы модели (например о просадке в 10%
капитала)
